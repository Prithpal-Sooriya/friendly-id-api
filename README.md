# test-project

Test Project to create a short friendly temp id that is less than bytes (so this can be used with legacy BLE advertising packet)

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.0. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
